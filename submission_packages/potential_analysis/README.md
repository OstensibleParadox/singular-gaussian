# Potential Analysis anonymous package

Journal: Potential Analysis
Publisher: Springer Nature
Official submission guidelines: https://link.springer.com/journal/11118/submission-guidelines
Official LaTeX support: https://www.springernature.com/gp/authors/campaigns/latex-author-support
Downloaded template: official_springer_nature_latex_Dec2024.zip
Template URL: https://cms-resources.apps.public.k8s.springernature.io/springer-cms/rest/v1/content/18782940/data/v12

Package files:
- manuscript.pdf: current anonymous compiled manuscript PDF.
- potential_analysis_anonymous_source.zip: flat LaTeX source package.
- official_springer_nature_latex_Dec2024.zip: official Springer Nature article template archive.

Source package notes:
- main.tex uses Springer Nature sn-jnl with sn-mathphys-num style.
- author/affiliation are anonymous placeholders.
- bibliography is supplied through paper1_blind.bbl and paper1_ref.bib.
- figures are flattened into source root; no subfolders inside source zip.
- sn-jnl.cls and sn-mathphys-num.bst are included.
- MSC placeholder: 47A64, 47B35, 60G15, 60H15.

Submission reminder:
- Springer says uploaded source files are compiled into a single PDF for peer review.
- Springer LaTeX support says Snapp source must compile with pdflatex and be compressed as zip.
- Springer LaTeX support recommends .bbl pasted/included when bibliography conversion is fragile.
